# Contributing

This is a personal toolbox. Issues and PRs are welcome, but reviewed when I
have time — sometimes that's a few days, sometimes longer.

If you're proposing a behavior change to one of the workflow skills
(`afk-issues`, `afk-army`, `uat`), please open an issue first describing the
problem you hit before writing the patch. The skills are tightly coupled by a
shared body template and label vocabulary; a change in one usually implies a
change in the other two.

If you're proposing a fix to a vendored skill (`grill-me`), the right place
is usually the upstream repo:

- grill-me → https://github.com/mattpocock/skills

We refresh from upstream periodically; fixes that land there land here too.
